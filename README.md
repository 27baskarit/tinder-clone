Custom URL Shortner

* [Express](https://expressjs.com/)
* [Mongoose](http://mongoosejs.com/)
* [React](https://reactjs.org/)

## Getting Started

```sh
# clone it
git clone https://github.com/27baskarit/tinder-clone.git

# Install dependencies
npm install

# Start  server
cd backend
node server.js

# Start  frontend
cd frontend
npm start
```
